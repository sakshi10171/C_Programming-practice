# C Array Programs

This folder contains **array-based programs in C**.  
These programs help in understanding array operations and logic building.

---

## 📂 Programs Included

| File Name | Description |
|----------|-------------|
| sum_array.c | Calculates sum of array elements |
| largest_array.c | Finds largest element in array |
| reverse_array.c | Reverses the array |
| linear_search.c | Searches an element using linear search |
| sort_array.c | Sorts array in ascending order using bubble sort |

---

## 📌 Concepts Covered
- Array input and output  
- Traversing arrays using loops  
- Searching techniques  
- Sorting algorithms (Bubble Sort)
- Basic problem-solving using arrays
